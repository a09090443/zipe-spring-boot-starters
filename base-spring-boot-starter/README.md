Spring boot starters
======================
## 說明
1. 該專案為客製化 Spring boot starters
2. 專案列表:
- base-spring-boot-starter: 基礎專案
- db-spring-boot-starter: 數據庫專案
- job-spring-boot-starter: 任務排程專案
- logon-spring-boot-starter: 登入控制專案
- web-service-spring-boot-starter: Cxf webservices 專案
- web-spring-boot-starter: web framework 專案
- example: 範例專案

## 開發環境
* OpenJDK 17
* Spring boot 3+
* Maven 3.8.4+
